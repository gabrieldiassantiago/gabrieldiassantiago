## 👤 Quem sou eu

> Comecei a codar aos **13 anos** criando interfaces pra servidores de jogos. Desde então, nunca separei código de estética.

Sou dev full-stack com um pé firme no design — o que significa que eu não só faço funcionar, mas me importo com como vai **sentir na mão de quem usa**. Trabalho com **React, Next.js, Node e TypeScript** no dia a dia. Quando a ideia precisa ir pro bolso das pessoas, entro com **React Native**.

No backend, o playground é **NestJS, Redis, BullMQ**, com bancos que variam de **PostgreSQL** até **MongoDB** quando o projeto pede. **Supabase e Firebase** entram pra prototipagem rápida sem perder a sanidade.

```diff
+ O que me move: tecnologia centrada em pessoas, não em features.
```
